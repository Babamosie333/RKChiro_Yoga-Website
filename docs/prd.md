# RK CHIRO Yog Centre Website Requirements Document

## 1. Website Overview
### 1.1 Website Name
RK CHIRO Yog Centre

### 1.2 Website Description\nA professional website for RK CHIRO Yog Centre, designed to showcase the center's chiropractic-based yoga programs, therapeutic skills, and provide easy contact access for potential clients.

### 1.3 Technical Requirements
- Frontend Technologies: HTML, CSS, JavaScript only
- No backend frameworks or additional languages
\n## 2. Website Structure\n
### 2.1 Page 1: Landing Page
- Welcome section introducing RK CHIRO Yog Centre
- Brief overview of the center's chiropractic yoga approach and therapeutic values
- Call-to-action buttons directing to Skills and Contact pages\n- Hero section with engaging visuals related to chiropractic therapy

### 2.2 Page 2: Skills Page
- Display of chiropractic-based yoga programs and therapeutic classes offered
- Chiropractor qualifications and expertise
- Benefits of chiropractic yoga practices for spinal health and alignment
- Therapeutic techniques and treatment approaches
- Photos showcasing chiropractic therapy sessions and techniques

### 2.3 Page 3: Contact Page
- Contact form for inquiries
- Center address and contact information
- Embedded Google Map: https://share.google/2Z0gef4XqAXmXJmhU
- Operating hours
- Social media links (if applicable)

## 3. Design Style

### 3.1 Color Scheme
- Primary colors: Calming teal (#008B8B) and soft purple (#9370DB) reflecting wellness and spirituality
- Secondary colors: Warm beige (#F5F5DC) for backgrounds, deep charcoal (#2F4F4F) for text
- Accent color: Soft gold (#DAA520) for call-to-action elements

### 3.2 Visual Details
- Rounded corners (8-12px border-radius) for cards and buttons to create a gentle, welcoming feel
- Subtle box shadows (04px 6px rgba(0,0,0,0.1)) for depth without harshness\n- Clean sans-serif fonts (e.g., 'Poppins' or 'Lato') for modern readability
- Smooth hover transitions (0.3s ease) on interactive elements\n- Minimalist icons in line style for navigation and features
\n### 3.3 Layout Approach
- Card-based layout for Skills page to organize different chiropractic programs and therapies
- Single-column responsive design for mobile-first approach
- Generous white space to maintain a calm, uncluttered aesthetic\n- Sticky navigation bar for easy page access
- Grid layout for testimonials or treatment schedules (if added later)
